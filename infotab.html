HTML

HTML veb səhifələrin strukturunu yaratmaq üçün istifadə olunan işarələmə dilidir. O, mətnləri, şəkilləri, linkləri və digər elementləri səhifədə düzgün şəkildə yerləşdirməyə imkan verir. HTML həmçinin CSS və JavaScriptlə birlikdə işləməyə imkan verir.

CSS

CSS veb səhifələrin görünüşünü və dizaynını formalaşdırmaq üçün istifadə olunan üslublandırma dilidir. O, rənglər, ölçülər, məsafələr, fontlar və yerləşmə kimi visual elementləri idarə edir.

JavaScript

JavaScript veb səhifələrə dinamika və interaktivlik əlavə edən proqramlaşdırma dilidir. O, istifadəçi ilə real vaxtda qarşlıqlı əlaqə qurmağa imkan verir və düymələr, animasiyalar, açılıb-bağlanan menyular kimi funksiyaları həyata keriçir. JavaScript həmçinin HTML və CSS-lə birlikdə işləyərək veb layihələrinin funksional və canlı görünməsini təmin edir.

Tailwand CSS

Tailwand CSS sürətli və effektiv şəkildə dizayn yaratmağa imkan verən utilit-əsaslı CSS frameword-dür. O, əvvəlcədən yazılmış siniflər vasitəsilə elementlərin ölçüsünü rəngini, məsafələrini vermək üçün istifadə edilir.

React JS

React JS interektiv və sürətli istifadəçi interfeysləri yaratmaq üçün istifadı olunan məhşur JavaScript kitabxanısıdır. O, komponent əsaslı qurulduğu üçün kodu kiçik və təkrar istifadə oluna bilən hissələrə bölməyə imkan verir.
